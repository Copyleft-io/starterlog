Factory

Overview

The factory service returns a $firebaseArray which is synchronized to the models data.

The controller depends on the factory service to bind model data to the views via $firebaseArray and $firebaseObject.

Note: the factory service is defined before the controller in app/index.html so that the factory module dependency is loaded before the controller.

Code Sample

'use strict';

app.factory("Models", function ModelFactory(FIREBASE_URL, $firebaseArray) {
    return function(){

    // snapshot of our data
    var ref = new Firebase(FIREBASE_URL + 'models');

    // returning synchronized array
    return $firebaseArray(ref);
  }
});

// log output to console for easier debugging
console.log('--> starterlog/app/models/models.service.js loaded');
